La función que permite calcular el valor absoluto de un número real es una función que recibe como parámetro de entrada un número real y retorna la distancia de ese valor al origen. La función valor absoluto en notación matemática se define como:

valorAbsoluto : \(\mathbb{R} \rightarrow \mathbb{R}\) $$􏰄(x) = \begin{cases} x, & \text{si $x \geq 0$;} \\ -x, & \text{en otro caso.}\end{cases}$$

## Condicional Sí (if): Ejemplo en C++ ```cpp #include <iostream> using namespace std; float valorAbsoluto(float x) { if(x >= 0) { return x; } else { return -x; } } int main() { float x = 0; float y = 0; cout << "Ingrese un numero: \t"; cin >> x; y = valorAbsoluto(x); cout << "El valor absoluto de " << x << " es: " << y << endl; return 0; } ```
## Condicional Sí (if): Ejemplo en Python ```python def valorAbsoluto(x): if x >= 0: return x else: return -x x = float(input("Ingrese un numero: \t")) y = valorAbsoluto(x) print("El valor absoluto de " + str(x) + " es: " + str(y)) ```
## Condicional Sí (if): Ejemplo en C++ ```cpp #include <iostream> using namespace std; float valorAbsoluto(float x) { return x >= 0 ? x : -x; } int main() { float x = 0; float y = 0; cout << "Ingrese un numero: \t"; cin >> x; y = valorAbsoluto(x); cout << "El valor absoluto de " << x << " es: " << y << endl; return 0; } ```

Condicional Sí (if): Ejemplo

Una función que permite determinar el máximo de dos números reales, se puede definir como:

maximo : \(\mathbb{R} \times \mathbb{R} \rightarrow \mathbb{R}\)

aquí se tienen dos casos, si el número \(a\) es mayor que \(b\) el valor máximo es \(a\); en otro caso se debe retornar \(b\). En notación matemática esto puede ser escrito de la siguiente forma:

maximo : \(\mathbb{R} \times \mathbb{R} \rightarrow \mathbb{R}\) $$􏰄(a, b) = \begin{cases} a, & \text{si $a > b$;} \\ b, & \text{en otro caso.}\end{cases}$$

## Condicional Sí (if): Ejemplo en C++ ```cpp #include <iostream> using namespace std; float maximo(float a, float b) { if(a > b) { return a; } else { return b; } } int main() { float a = 0; float b = 0; float y = 0; cout << "Ingrese un numero: \t"; cin >> a; cout << "Ingrese un numero: \t"; cin >> b; y = maximo(a, b); cout << y << " es el mayor de los 2 numeros." << endl; return 0; } ```
## Condicional Sí (if): Ejemplo en Python ```python def maximo(a, b): if a > b: return a else: return b a = float(input("Ingrese un numero: \t")) b = float(input("Ingrese un numero: \t")) y = maximo(a, b) print(str(y) + " es el mayor de los 2 numeros.") ```
## Condicional Sí (if): Usando el operador ternario
## Condicional Sí (if): Ejemplo en C++ ```cpp #include <iostream> using namespace std; float maximo(float a, float b) { return a > b ? a : b; } int main() { float a = 0; float b = 0; float y = 0; cout << "Ingrese un numero: \t"; cin >> a; cout << "Ingrese un numero: \t"; cin >> b; y = maximo(a, b); cout << y << " es el mayor de los 2 numeros." << endl; return 0; } ```
## Condicional Sí (if): Ejemplo en Python ```python def maximo(a, b): return a if a > b else b a = float(input("Ingrese un numero: \t")) b = float(input("Ingrese un numero: \t")) y = maximo(a, b) print(str(y) + " es el mayor de los 2 numeros.") ```

Estructura condicional sin opción alternativa: Ejemplo

Se puede definir una función que permita calcular la operación condicional de un par de variables booleanas y que retorna el resultado de operar los valores mediante un condicional, de la siguiente manera:

condicional : \(\mathbb{B} \times \mathbb{B} \rightarrow \mathbb{B}\)

aquí se tienen dos casos, primero, si el antecedente es verdadero y el consecuente es falso, entonces el resultado de aplicar el condicional es falso, para cualquier otro caso el condicional es verdadero.

Estructura condicional sin opción alternativa: Ejemplo

En notación matemática esto puede ser escrito de la siguiente manera:

condicional : \(\mathbb{B} \times \mathbb{B} \rightarrow \mathbb{B}\) $$􏰄(p, q) = \begin{cases} F, & \text{si $\xi(p) = V \land \xi(q) = F;$} \\ V, & \text{en cualquier otro caso.}\end{cases}$$

## Estructura condicional sin opción alternativa: Ejemplo en C++ ```cpp #include <iostream> using namespace std; bool condicional(bool p, bool q) { if(p == true && q == false) { return false; } return true; } int main() { bool p = false; bool q = false; bool y = false; cout << "Ingrese un booleano: \t"; cin >> p; cout << "Ingrese un booleano: \t"; cin >> q; y = condicional(p, q); cout << "El condicional entre " << p; cout << " y " << q; cout << " es: " << y << endl; return 0; } ```
## Estructura condicional sin opción alternativa: Ejemplo en Python ```python def condicional(p, q): if p == True and q == False: return False return True p = bool(int(input("Ingrese un booleano: \t"))) q = bool(int(input("Ingrese un booleano: \t"))) y = condicional(p, q) print("El condicional entre " + str(p), end="") print(" y " + str(q), end="") print(" es: " + str(y)) ```
## Estructura condicional sin opción alternativa: Ejemplo en C++ ```cpp #include <iostream> using namespace std; bool condicional(bool p, bool q) { return (p == true && q == false) ? false : true; } int main() { bool p = false; bool q = false; bool y = false; cout << "Ingrese un booleano: \t"; cin >> p; cout << "Ingrese un booleano: \t"; cin >> q; y = condicional(p, q); cout << "El condicional entre " << p; cout << " y " << q; cout << " es: " << y << endl; return 0; } ```
## Estructura condicional sin opción alternativa: Ejemplo en Python ```python def condicional(p, q): return False if (p == True and q == False) else True p = bool(int(input("Ingrese un booleano: \t"))) q = bool(int(input("Ingrese un booleano: \t"))) y = condicional(p, q) print("El condicional entre " + str(p), end="") print(" y " + str(q), end="") print(" es: " + str(y)) ```
## Condicionales Enlazados
## Condicionales Enlazados: Definición Otra de las opciones para utilizar una estructura condicional es la de enlazar varias estructuras condicionales, de tal manera que solamente se pueda ejecutar un grupo de instrucciones dependiendo de cual de las opciones se evaluá verdadero. De la misma manera que en el caso del condicional tradicional la parte alternativa del final es opcional.
## Condicionales Enlazados: Esquema en C++ ```cpp #include <iostream> using namespace std; int main() { // Instrucciones if(//Condicion 1) { // Instruccion 1 cuando la condicion 1 se cumple // Instruccion 2 cuando la condicion 1 se cumple // ... // Instruccion m cuando la condicion 1 se cumple } else if(//Condicion 2) { // Instruccion 1 cuando la condicion 2 se cumple // Instruccion 2 cuando la condicion 2 se cumple // ... // Instruccion n cuando la condicion 2 se cumple } // ... else if(//Condicion i) { // Instruccion 1 cuando la condicion i se cumple // Instruccion 2 cuando la condicion i se cumple // ... // Instruccion o cuando la condicion i se cumple } // ... else if(//Condicion n-1) { // Instruccion 1 cuando la condicion n-1 se cumple // Instruccion 2 cuando la condicion n-1 se cumple // ... // Instruccion p cuando la condicion n-1 se cumple } else { // Instruccion 1 cuando ninguna de las condiciones se cumple // Instruccion 2 cuando ninguna de las condiciones se cumple // ... // Instruccion q cuando ninguna de las condiciones se cumple } // Mas instrucciones return 0; } ```
## Condicionales Enlazados: Esquema en Python ```python # Instrucciones if #Condicion 1: # Instruccion 1 cuando la condicion 1 se cumple # Instruccion 2 cuando la condicion 1 se cumple # ... # Instruccion m cuando la condicion 1 se cumple elif #Condicion 2: # Instruccion 1 cuando la condicion 2 se cumple # Instruccion 2 cuando la condicion 2 se cumple # ... # Instruccion n cuando la condicion 2 se cumple # ... elif #Condicion i: # Instruccion 1 cuando la condicion i se cumple # Instruccion 2 cuando la condicion i se cumple # ... # Instruccion o cuando la condicion i se cumple # ... elif #Condicion n-1: # Instruccion 1 cuando la condicion n-1 se cumple # Instruccion 2 cuando la condicion n-1 se cumple # ... # Instruccion p cuando la condicion n-1 se cumple else: # Instruccion 1 cuando ninguna de las condiciones se cumple # Instruccion 2 cuando ninguna de las condiciones se cumple # ... # Instruccion q cuando ninguna de las condiciones se cumple # Mas instrucciones ```

Condicionales Enlazados: Ejemplo

Para un procedimiento que permite comparar dos números reales (\(a, b\)), se pueden tener las siguientes condiciones:

  • Si el número \(a\) es mayor que \(b\), el programa deberá indicar que \(a\) es mayor que \(b\).
  • Si el número \(b\) es mayor que \(a\), el programa deberá indicar que \(b\) es mayor que \(a\).
  • Si el número \(a\) es igual al número \(b\), el programa deberá indicar que ambos números son iguales.

Condicionales Enlazados: Ejemplo

Podemos escribir lo anterior de la siguiente forma:

$$\begin{cases} a > b, & \text{si $a > b$;} \\ b > a, & \text{si $b > a$;} \\ a = b, & \text{en otro caso.}\end{cases}$$

## Condicionales Enlazados: Ejemplo en C++ ```cpp #include <iostream> using namespace std; void comparar(float a, float b) { if(a > b) { cout << "a es mayor que b." << endl; } else if (b > a) { cout << "b es mayor que a." << endl; } else { cout << "Los numeros son iguales." << endl; } } int main() { float a = 0; float b = 0; cout << "Ingrese el valor de a: \t"; cin >> a; cout << "Ingrese el valor de b: \t"; cin >> b; comparar(a, b); return 0; } ```
## Condicionales Enlazados: Ejemplo en Python ```python def comparar(a, b): if a > b: print("a es mayor que b.") elif b > a: print("b es mayor que a.") else: print("Los numeros son iguales.") a = float(input("Ingrese el valor de a: \t")) b = float(input("Ingrese el valor de b: \t")) comparar(a, b) ```
## Condicionales Enlazados: Ejemplo Si un cliente lleva más de 5 productos del mismo tipo le realizan un descuento del 5%. Si lleva más de 10 productos del mismo tipo le realizan un descuento del 10%. Si lleva más de 20 productos del mismo tipo le realizan un descuento del 20%. Realizar un programa que dado el número de productos y el precio del producto determine el valor a pagar por el cliente.

Condicionales Enlazados: Ejemplo

Entonces tenemos que:

pagoFinal = \(\mathbb{Z} \times \mathbb{R} \rightarrow \mathbb{R}\)

$$(n, precio) \rightarrow \begin{cases} n * precio, & \text{si $n < 5$;} \\ n * precio * 0.95, & \text{si $5 < n \leq 10$;} \\ n * precio * 0.9, & \text{si $10 < n \leq 20$;} \\ n * precio * 0.8, & \text{en otro caso.}\end{cases}$$

## Condicionales Enlazados: Ejemplo en C++ ```cpp #include <iostream> using namespace std; float pagoFinal(int n, float precio) { if(n <= 5) { return n * precio; } else if(5 < n && n <= 10) { return n * precio * 0.95; } else if(10 < n && n <= 20) { return n * precio * 0.9; } else { return n * precio * 0.8; } } int main() { int n = 0; float precio = 0; float y = 0; cout << "Ingrese el numero de productos: \t"; cin >> n; cout << "Ingrese el precio del producto: \t"; cin >> precio; y = pagoFinal(n, precio); cout << "El valor a pagar es: " << y << endl; return 0; } ```
## Condicionales Enlazados: Ejemplo en Python ```python def pagoFinal(n, precio): if n <= 5: return n * precio elif 5 < n and n <= 10: return n * precio * 0.95 elif 10 < n and n <= 20: return n * precio * 0.9 else: return n * precio * 0.8 n = int(input("Ingrese el numero de productos: \t")) precio = float(input("Ingrese el precio del producto: \t")) y = pagoFinal(n, precio) print("El valor a pagar es: " + str(y)) ```

## Estructura Switch
## Estructura Switch: Esquema en C++ ```cpp #include <iostream> using namespace std; int main() { // Instrucciones switch(//Variable) { case //Valor 1: // Instruccion 1 cuando la variable toma el 1er valor // Instruccion 2 cuando la variable toma el 1er valor // ... // Instruccion m cuando la variable toma el 1er valor break; case //Valor 2: // Instruccion 1 cuando la variable toma el 2do valor // Instruccion 2 cuando la variable toma el 2do valor // ... // Instruccion n cuando la variable toma el 2do valor break; // ... case //Valor i: // Instruccion 1 cuando la variable toma el i-esimo valor // Instruccion 2 cuando la variable toma el i-esimo valor // ... // Instruccion o cuando la variable toma el i-esimo valor break; // ... case //Valor n-1: // Instruccion 1 cuando la variable toma el n-1-esimo valor // Instruccion 2 cuando la variable toma el n-1-esimo valor // ... // Instruccion p cuando la variable toma el n-1-esimo valor break; default: // Instruccion 1 cuando la variable toma ninguno de los valores // Instruccion 2 cuando la variable toma ninguno de los valores // ... // Instruccion q cuando la variable toma ninguno de los valores break; } // Mas instrucciones return 0; } ```
## Estructura Switch: Ejemplo Dado un caracter, determinar si este es una vocal minúscula.
## Estructura Switch: Ejemplo en C++ ```cpp #include <iostream> using namespace std; bool esVocal(char c) { return c == 'a' || c == 'e' || c == 'i' || c == 'o' || c == 'u'; } int main() { char c = 0; cout << "Ingrese un caracter: \t"; cin >> c; cout << esVocal(c) << endl; return 0; } ```
## Estructura Switch: Ejemplo en Python ```python def esVocal(c): return c == 'a' or c == 'e' or c == 'i' or c == 'o' or c == 'u' c = str(input("Ingrese un caracter: \t")) print(str(esVocal(c))) ```
## Estructura Switch: Ejemplo en C++ ```cpp #include <iostream> using namespace std; bool esVocal(char c) { switch(c) { case 'a': return true; break; case 'e': return true; break; case 'i': return true; break; case 'o': return true; break; case 'u': return true; break; default: return false; break; } } int main() { char c = 0; cout << "Ingrese un caracter: \t"; cin >> c; cout << esVocal(c) << endl; return 0; } ```
## Estructura Switch: Ejemplo Cuando en una instrucción switch varios casos se tratan de la misma forma, entonces estos se pueden agrupar en uno o varios casos, como se muestra a continuación:
## Estructura Switch: Ejemplo en C++ ```cpp #include <iostream> using namespace std; bool esVocal(char c) { switch(c) { case 'a': case 'e': case 'i': case 'o': case 'u': return true; break; default: return false; break; } } int main() { char c = 0; cout << "Ingrese un caracter: \t"; cin >> c; cout << esVocal(c) << endl; return 0; } ```
## Estructura Switch: Ejemplo Es usual que las instructiones que se ejecutan dentro de los casos de una instrucción switch se retornen directamente cuando se calculen dentro de una función, para estos casos, en vez de dejar de ejecutar el switch con un break, se puede retornar el valor directamente mediante el uso de la instrucción return.
## Estructura Switch: Ejemplo en C++ ```cpp #include <iostream> using namespace std; bool esVocal(char c) { switch(c) { case 'a': case 'e': case 'i': case 'o': case 'u': return true; default: return false; } } int main() { char c = 0; cout << "Ingrese un caracter: \t"; cin >> c; cout << esVocal(c) << endl; return 0; } ```
